# ARDUboard

Это новая, созданная нами, плата Arduino на основе процессора ATmega328P, является компактной, удобной и автономной средой для как разработки и так обучения, где одна плата заменяет целые наборы электроники.
Визиткой данной платы есть кардинально усиленные питающие цепи, возможность работы от 24V, переключатель логических уровней Arduino между +5V и +3.3V и универсальный microUSB вместо громоздкого USB type-B, встроенный полномостовой драйвер 24В на 3.5А  для любых мощных нагрузок и еще очень много полезных функций…
